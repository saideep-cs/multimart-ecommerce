# E-commerce Experience using Contentstack

This project demonstrates a simple end-to-end e-commerce digital experience built using Contentstack as part of the internal bootcamp learning initiative.

The goal of this project is to learn by doing and showcase how multiple Contentstack products can be used together to power a real-world use case.

## Overview
The application fetches product and page content from Contentstack CMS, uses DAM for asset management, triggers transactional emails using Automate, and is deployed using Contentstack Launch.

## Contentstack Products Used
- **Content Management System (CMS)** – for managing product and page content
- **Digital Asset Management (DAM)** – for managing and serving images and media
- **Automate** – for triggering order confirmation emails
- **Launch** – for deploying and hosting the application

## Key Features
- Product listing and detail pages powered by CMS content
- Assets fetched dynamically from DAM
- Order placement flow with email notification
- Live deployment using Contentstack Launch

## Tech Stack
- React
- Contentstack REST APIs
- Contentstack Automate
- Contentstack Launch
